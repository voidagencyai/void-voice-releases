# Void Voice — Releases

Este repositorio aloja los artefactos firmados y el manifiesto de auto-actualización de **Void Voice**.

La app no es open source. El código fuente es privado; aquí solo se publican los instaladores y el `latest.json` que usa el updater integrado.

## Descargar

Ve a [Releases](https://github.com/voidagencyai/void-voice-releases/releases) para descargar la última versión.

- **macOS** (Apple Silicon): `.dmg`
- **Windows** (x64): `-setup.exe`
